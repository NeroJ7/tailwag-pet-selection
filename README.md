# TailWag (摇尾精选) - 宠物用品优选网站

## 1. 项目定位
提供高品质、经过筛选的宠物用品，通过专业评测和用户口碑，解决养宠人士“选品难”的问题。品牌口号：“每一次摇尾，都是对我们的信任。”

## 2. 核心功能 (已完成 ✅)
- [x] **品牌视觉系统**: 包含官方 Logo 与 橙色系 UI 规范。
- [x] **首页 (Home)**: 包含精选模块、信任价值背书、20款热门商品展示。
- [x] **选品流程 (Selection Process)**: 详细展示品牌筛选标准。
- [x] **商品详情 (Product Detail)**: 深度展示优选理由、技术规格与用户评价。
- [x] **购物车与结算 (Cart)**: 支持数量调整与订单摘要。
- [x] **会员中心 (Login)**: 用户登录与会员权益展示。
- [x] **品牌入驻 (B2B)**: 专业的供应商招募申请页面。

## 3. 技术架构
- **前端**: Next.js (React) - 提供极致的加载速度与 SEO 效果。
- **样式**: Tailwind CSS - 响应式优先，完美适配手机与电脑。
- **数据**: JSON-Driven architecture - 便于后续无缝对接后端数据库。

## 4. 部署与上线指南
您可以按照以下步骤将网站发布至公网：
1. **GitHub 托管**: 将 `pet-selection-site` 文件夹推送到 GitHub 仓库。
2. **Vercel 部署 (推荐)**: 关联 GitHub 仓库，一键实现全自动部署。
3. **域名绑定**: 在 Vercel 后台绑定您购买的域名（如 `tailwag.pet`）。
